(В этом файле представлены не столь баги, сколь вопросы
к написанию документации и описанию функционала методов класса.)

1. to_list:
строка, состоящая из пробела распознаётся в качестве абсолютно пустой строки,
в связи с чем результатом работы метода является пустой список.
Ожидаемый результат: если разделителем выступает НЕ пробел, результатом работы метода должен являться
список, состоящий из одной строки, содержащей пробел [" "]. - Проверка №21

2. test_starts_with:
Необходимо уточнение в описании, что метод может проверять не только один символ,
т.е. с помощью него возможно проверить, начинается ли строка с какой-либо подстроки. - Проверка №37

3. test_end_with:
Необходимо уточнение в описании, что метод может проверять не только один символ,
т.е. с помощью него возможно проверить, заканчивается ли строка какой-либо подстроки. - Проверка №44

4. is_empty:
Строка, содержащая только пробелы, распознаётся в качестве абсолютно пустой строки,
в связи с чем результатом работы метода является неверное булевое значение. - Проверка №52

5. list_to_string:
Разделителем по умолчанию является не просто запятая, как сказано в описании метода,
а запятая с пробелом после неё.
